# meta-archives.xyz

아카이브를 비평하는 아카이브

## About

디지털 아카이브의 메타데이터, 기술 구조, 분류 방식이  
어떤 세계관과 기억 방식을 전제하는지 드러냅니다.

→ [meta-archives.xyz](https://meta-archives.xyz)

## 구조

```
meta-archives/
├── index.html              # 메인 페이지
├── about.html              # About 페이지
├── posts/                  # 개별 글 HTML
├── content/posts/          # 비평 글 원본 (Markdown)
├── data/posts/             # 메타데이터 (YAML)
├── assets/                 # CSS, 이미지, 파비콘
└── docs/                   # 프로젝트 문서
```
